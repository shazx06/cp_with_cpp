1.Find the largest subarray formed by consecutive integers
    Input:  { 2, 0, 2, 1, 4, 3, 1, 0 }
    Output: The largest subarray is { 0, 2, 1, 4, 3 } 

2.Find maximum profit earned by buying and selling shares any number of times
    input: Stock Prices: {1, 5, 2, 3, 7, 6, 4, 5}
    output : 10
    
    explanation:
    Buy on day 1 and sell on day 2
    Buy on day 3 and sell on day 5
    Buy on day 7 and sell on day 8
    
    input : stock prices : {10, 8, 6, 5, 4, 2}
    output : 0
    
3.Given an integer array, find the maximum product subarray. In other words, find a subarray that has the maximum product of its elements.

    Input:  { -6, 4, -5, 8, -10, 0, 8 }
    Output: The maximum product subarray is {4, -5, 8, -10} having product 1600.

    Input:  { 40, 0, -20, -10 } 
    Output: The maximum product subarray is {-20, -10} having product 200
